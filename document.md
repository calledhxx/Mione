# 概要
Mione程式語言 (The Mione Programming Language) 是一款高級現代化的程式語言。
本語言設計採用了與Shell相同的概念：一切都是命令。並且能透過多個程式句的組合來實現複雜的要求。

除此之外，Mione語言還設定了「詞性」的語法系統，使初學者能夠更輕易的上手該門程式語言。
詞性包括：Head (標頭), Prompt (提詞), Symbol (符號), Variable (變數), Value (立即值), Weld (連接詞) 共六種。
